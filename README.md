# 🌐 Responsive Landing Page

## 📌 Project Description

This project is a modern and fully responsive landing page designed to provide an optimal viewing experience across all devices, including mobile phones, tablets, and desktops. The layout adapts smoothly to different screen sizes to ensure a consistent and user-friendly interface.

## 🚀 Features

* Fully responsive design
* Clean and modern UI
* Smooth navigation
* Interactive elements
* Mobile-friendly layout

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## ▶️ How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click the `index.html` file.
4. The landing page will open in your browser.
